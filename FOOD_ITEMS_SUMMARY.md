## üöÄ **HOW TO USE**

### **Import the Food Items:**
```javascript
import { allFoodItems, getItemsByMeal, getItemsByCategory } from './all-food-items.js';

// Get all items
const allFood = allFoodItems;

// Get items by meal
const breakfastItems = getItemsByMeal('breakfast');
const lunchItems = getItemsByMeal('lunch');
const dinnerItems = getItemsByMeal('dinner');

// Get items by category
const indianFood = getItemsByCategory('indian');
const italianFood = getItemsByCategory('italian');
```

### **Filter by Multiple Criteria:**
```javascript
import { getItemsByMealAndCategory } from './all-food-items.js';

// Get Indian breakfast items
const indianBreakfast = getItemsByMealAndCategory('breakfast', 'indian');

// Get all lunch items
const allLunch = getItemsByMealAndCategory('lunch', 'all');
```
